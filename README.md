# AI 작업실 MVP

Claude, GPT, Gemini, Cursor, Figma, Make, Zapier 같은 AI 도구로 작업하는 사람들이 함께 머무는 온라인 작업실 MVP입니다.

## 포함된 기능

- 작업방 목록과 방별 8명/12명 인원 제한
- 내 작업 상태, 오늘의 목표, 사용 도구 저장
- 방 입장/퇴장에 가까운 방 전환 경험
- 방 안 도움 요청 카드
- 작업 채팅
- 카테고리별 Q&A
- 결과물 쇼케이스
- 커피챗 후보 매칭
- 브라우저 `localStorage` 기반 데모 데이터 저장

## 실행
